# Session Update

## Description
Update session file after completing a step.

## Command
Edit session file to mark step as completed.

## Format
Update the "Estado" column from `⏳` to `✅`:

```markdown
| Paso | Estado | Descripción |
|------|--------|-------------|
| 1 | ✅ | start - Buscar/Crear sesión |
| 2 | ✅ | INIT - Mostrar rol, reglas |
```

## Timing
Update after each step completion.
